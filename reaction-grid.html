How to Play

  1. Read each clue — compounds are given in words, not formulas.
  2. Find each compound's formula in the reference table below.
  3. Enter coefficients (1–9) so both sides of each reaction balance — equal atoms on each side.
  4. Orange cells are shared between two reactions. One number serves both.
  5. Press Check when every reaction is balanced.
  6. Expert only: formula labels are hidden — type each compound's formula in the dashed cell above its coefficient.
